Develop Apps and Capabilities for your Android 4.X device
  • 1. Download Android client from GitHub.
  • 2. Open the client in Eclipse (Android development version).
  • 3. Create a new package for you capability e.g. myNewCapability.
  • 4. Add new class to your capability, and name like you named the package, like MyNewCapability.java
  • 5. Each Android capability must have initCapability(Context context) function defined. (You can also copy code from some existing capability e.g. TalkingCapability and use it as a basis).
  • 6. If you need a Activity, remember to define it in the Manifest.xml file (just like with regular Android app).
  • 7. Deploy and test your app as you would do with regular Android app.
